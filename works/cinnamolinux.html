CinnamoLinux

A fan made Cinnamoroll (One of the mascot of Sanrio) themed
Linux system based on Xubuntu (16.04 for Ver1, 18.04 for Ver.2)
Since this is mainly for China so you'll see a lot of Chinese software.
Cinnamoroll is owned by Sanrio. Co. Ltd (Japan)

A few days after I made Warma Linux Ver.1, I became obsessed with Cinnamoroll from Sanrio.
His cuteness completely made me fall in love. And he sooner became one of my favourite character.
My crush on him sooner led to my idea of making a Cinnamoroll themed Linux. And that is it, CinnamoLinux.

Known Bugs

Ver.1:

  1. Uncustomized Default Theme.
  2. Only contains English language, so you'll have to set language again after installing. (If you're not going to use English)
  3. MBR users will have to use boot-repair to fix boot after installing.
  4. Some program won't work.

Ver.2:

  1. Both MBR and UEFI user will have to use boot-repair to fix boot after installing.
  2. Only contains English and Simplified Chinese language so...
  3. Chinese IME have issue working in live environment.
